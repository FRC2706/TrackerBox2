TrackerBox2
===========

A camera-based object tracker for use in the FIRST Robotics Competition

This project is being worked on by Mike Ounsworth and Alex Huctwith on FRC team 3710.


## Dependencies

Linux packages that need to be installed:
  - libyaml-cpp0.3-dev
  - libboost-dev
  - libopencv-dev
